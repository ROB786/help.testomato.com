Testomato REST API

Testomato REST API#

Testomato REST API provides simple interfaces for Testomato functionality. To use a REST API, your application makes an HTTP request and parses the JSON formatted response.

/api/project/*#

MethodNameEntrypointDescription
GETProject/api/project/[id]Getting basic Project information
GETStarting/api/project/[id]/startrun all checks in Project
GETResults/api/project/[id]/job/[jobid]/resultsReturn results of selected job
GETSimple Status/api/project/[id]/simple-statusSimplify project status
GETStatus/api/project/[id]/statusFull project status
GETResponse times/api/project/[id]/responseTimesResponse times in select time period
GETUsers/api/project/[id]/responseTimesProject Users
GETGroup/api/project/[id]/responseTimesProject Users

/api/user/*#

MethodNameEntrypointDescription
GETProject/api/user/[id]/projectsGetting list of user projects
