Areas

#Project Areas info

#API Endpoint

API endpoint
GET /api/areas/[id]?apikey=[key]

#Request params

ParameterDescriptionMandatory
idProject IDrequired
apikeyAn API key unique to each projectrequired

#Call example

Example CURL call
curl https://www.testomato.com/api/project/5e95ddf7a39e5a44428b4581/areas?apikey=4PJAOwTK
