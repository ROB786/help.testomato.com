Project Start

#API Endpoint

API endpoint
GET /api/project/[id]/start?apikey=[key]

#Request params

ParameterDescriptionMandatory
idYour project IDrequired
apikeyAn API key unique to each projectrequired

#Call example

Example CURL call
curl "https://www.testomato.com/api/project/5e95ddf7a39e5a44428b4581/start?apikey=4PJAOwTK"

#Response

Example JSON response
{
"projectId": "5e95ddf7a39e5a44428b4581",
"jobId": "5e9602e7a39e5a46428b457f",
"start": "ok",
"results": "https:\/\/www.testomato.com\/api\/project\/5e95ddf7a39e5a44428b4581\/job\/5e9602e7a39e5a46428b457f\/results?apikey=4PJAOwTK"
}

#Response Parameters

ParameterDescription
projectIdA unique project ID
jobIdA unique ID for each project job
startok means your tests have successfully started
resultsThe URL where you can view your results
