Project

#API Endpoint

API endpoint
GET /api/project/[id]?apikey=[key]

#Request params

ParameterDescriptionMandatory
idYour project IDrequired
apikeyAn API key unique to each projectrequired

#Call example

Example CURL call
curl https://www.testomato.com/api/project/5e95ddf7a39e5a44428b4581?apikey=4PJAOwTK
