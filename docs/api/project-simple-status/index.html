Project Simple Status

#API Endpoint

GET /api/project/[id]/simple-status?apikey=[key]

#Request params

ParameterDescriptionMandatory
idYour project IDrequired
apikeyAn API key unique to each projectrequired

#Call example

Example CURL call
curl https://www.testomato.com/api/project/5e95ddf7a39e5a44428b4581/simple-status?apikey=4PJAOwTK
